# Iris Flower Prediction 🌸

Machine Learning project using **Scikit-learn** and **Streamlit** to predict the species of an Iris flower.

## Technologies

- Python
- Scikit-learn
- Pandas
- Streamlit
- Joblib

## How it works

The model uses the following flower measurements:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The trained model predicts one of three species:

- Setosa
- Versicolor
- Virginica
